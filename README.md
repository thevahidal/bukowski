# Bukowski
A human typewriter

[Demo](https://bukowski-typewriter.netlify.app/)

![bukowski-screenshot.png](/docs/bukowsi-screenshot.png)

## Development

Clone this repo

```
git clone https://github.com/thevahidal/bukowski.git
cd bukowski
```

Install dependencies and run it

```
npm install
npm run dev
```
